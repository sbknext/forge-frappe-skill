# forge-frappe-skill

Frappe/ERPNext Agent Skills for Claude Code — production-grade patterns for custom app development, DocTypes, hooks, background jobs, service layers, permissions, and ERPNext extensions.

## Install
```bash
npx skills add https://github.com/sbknext/forge-frappe-skill/frappe-app-development
```

## Skills
- **frappe-app-development** — Scaffold and architect custom Frappe apps with ElasticRun-grade patterns
