A reusable pre-trip checklist covering bookings, documents, packing, and day-0 logistics.

## When to use
Start T-30 days for international/long trips; T-7 for domestic weekend trips.

## Checklist
```markdown
### T-30 (International) / T-7 (Domestic)
- [ ] Flights / trains booked + confirmation saved offline
- [ ] Accommodation confirmed (address + check-in time saved)
- [ ] Passport/ID validity checked (min 6 months for international)
- [ ] Visa applied / e-Visa printed if needed
- [ ] Travel insurance purchased
- [ ] International SIM / roaming plan activated (international)
- [ ] Currency exchanged or Forex card loaded

### T-3
- [ ] Web check-in done
- [ ] Boarding pass saved offline
- [ ] Local transport from airport figured out
- [ ] Emergency contacts noted (hotel, embassy, local friend)
- [ ] Important docs photographed and saved to cloud

### T-1 (Packing)
- [ ] Clothes (N+1 days, where N = trip length)
- [ ] Phone charger + adapter (if international)
- [ ] Medicines (personal + basic: paracetamol, antacid, ORS)
- [ ] Earphones
- [ ] Power bank (charged)
- [ ] Snacks for travel day
- [ ] All original documents in one envelope

### Day 0
- [ ] Set home alarm / lock check
- [ ] Notify neighbour if gone >3 days
- [ ] Forward important mail if needed
- [ ] Switch off AC, geyser, non-essential power
```

## Tips
- India context: IRCTC tickets on DigiLocker as backup if your phone dies.
- Take a photo of your hotel exterior and address in local language — useful for cab drivers.
- For mountain trips: add altitude sickness medication to the T-3 checklist.
