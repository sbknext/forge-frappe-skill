Adapt the OKR framework for personal life with 3 objectives and 2-3 measurable key results each.

## When to use
At the start of each quarter (Jan, Apr, Jul, Oct). Takes ~45 minutes.

## Framework
- **Objective**: Qualitative direction. Inspiring. "Get physically strong."
- **Key Result**: Measurable outcome by end of quarter. "Complete 30 gym sessions." Not tasks.
- Max 3 objectives. Max 3 KRs per objective. Less is more.

## Steps
1. Reflect on last quarter — what moved, what stalled, what changed in life context?
2. Identify 3 areas where you want real progress this quarter.
3. For each area, write one inspiring objective.
4. For each objective, write 2-3 binary or numerical KRs.
5. Assign a mid-quarter check-in date.
6. Score at end of quarter (0.0–1.0). Target is 0.6–0.7, not 1.0.

## Template
```
Q___ OKRs — [Year]

O1: [Objective 1]
  KR1.1: [Metric] from X to Y by [date]
  KR1.2: [Metric] ≥ N by [date]

O2: [Objective 2]
  KR2.1:
  KR2.2:

O3: [Objective 3]
  KR3.1:
  KR3.2:

Mid-quarter check-in: ____
End-of-quarter score date: ____

### End of Quarter Scores
| KR | Target | Actual | Score |
|----|--------|--------|-------|
| 1.1 | | | |
| 1.2 | | | |
...
Overall: ___
```

## Tips
- Personal OKRs should span multiple life areas: health, relationships, finance, learning, one other.
- A score of 1.0 means you set easy KRs. 0.3 means too ambitious. 0.6-0.7 is ideal.
- Keep OKRs visible — phone wallpaper, sticky note on monitor, or weekly review template.
