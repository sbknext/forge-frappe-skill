A master list of documents every adult should have accessible, current, and backed up.

## When to use
Once to audit your document status; review annually or after major life events (marriage, new property, job change).

## Checklist
```markdown
### Identity
- [ ] Aadhaar (physical card + DigiLocker)
- [ ] PAN card (physical + soft copy)
- [ ] Passport (valid, not expiring within 6 months if you travel)
- [ ] Voter ID
- [ ] Driving licence

### Financial
- [ ] Bank account passbooks / statements
- [ ] Demat account statement
- [ ] Income tax returns (last 3 years, downloaded from IT portal)
- [ ] EPF/PPF account number and nominee details
- [ ] Insurance policies (term, health, vehicle) — soft copies
- [ ] Loan statements (home, vehicle, personal) — latest

### Property / Agreements
- [ ] Rent agreement (current, signed)
- [ ] Property documents (if owner) — registered copy
- [ ] Vehicle RC book + insurance

### Medical
- [ ] Blood group card or note
- [ ] Allergy and current medication list
- [ ] Health insurance card (physical + soft copy)
- [ ] Emergency contact list (printed, not just on phone)

### Backup status
- [ ] All docs photographed and uploaded to personal cloud folder
- [ ] Family member knows where to find the folder
- [ ] Folder access shared with one trusted family member
```

## Tips
- India context: DigiLocker is the government's official platform — Aadhaar, PAN, and DL are available there and legally valid.
- For shared documents (joint property, spouse's policy), both parties should have copies.
- Review this list annually; add a reminder to your calendar.
