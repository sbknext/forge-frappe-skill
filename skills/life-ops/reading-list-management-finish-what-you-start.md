A simple system to track, prioritise, and actually finish books instead of accumulating a guilt pile.

## When to use
Whenever your "to-read" list is longer than you can realistically read this year.

## The problem
Unmanaged reading lists grow faster than reading speed. The fix is a ruthless three-tier system.

## Steps
1. List every book you want to read this year.
2. Sort into three buckets: **Active** (reading now, max 2), **Queue** (next 5), **Someday** (everything else).
3. Never add to Active unless one leaves.
4. At the start of each month, move one book from Queue to Active.
5. When a Someday book hasn't been promoted in 12 months, delete it — your interests have changed.

## Template
```
Reading system — [Year]

### Active (max 2)
| Title | Author | Started | Target finish | Progress |
|-------|--------|---------|---------------|----------|
| | | | | |

### Queue (next 5)
1.
2.
3.
4.
5.

### Someday (no pressure)
- [Title] — [Why it interests me]
- ...

### Finished this year
| Title | Finished | Rating/1-line takeaway |
|-------|----------|------------------------|
| | | |
```

## Tips
- Two active books: one non-fiction (read in the morning with focus) and one fiction (read at night before sleep).
- The one-line takeaway after finishing is more valuable than highlights — forces synthesis.
- India context: Kindle Unlimited is ₹170/month and worth it if you read 2+ books/month.
