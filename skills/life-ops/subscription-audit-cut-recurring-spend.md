Find and cancel subscriptions you've forgotten about in a 30-minute annual purge.

## When to use
Once a year — best done in January or after a bank statement review flags unfamiliar recurring charges.

## Steps
1. Pull last 3 months of bank/card statements.
2. Search for recurring charges: same amount, same merchant, multiple months.
3. List every subscription found (see template).
4. For each, answer: Did I use this in the last 30 days? Would I miss it if cancelled today?
5. Cancel anything that scores NO on both. Keep anything YES/YES. For YES/NO — pause or downgrade.
6. Set a calendar reminder 3 days before each free trial ends.


## Tips
- India context: Check UPI Autopay mandates via your bank app — they're separate from card subscriptions.
- Google Pay → Manage Payments; Paytm → My Subscriptions; PhonePe → Recurring Payments.
- App Store (iOS/Android) subscriptions are separate — check in device Settings > Subscriptions.
