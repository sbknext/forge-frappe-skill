Reduce email to zero unread daily using a 4-action triage: Do, Delegate, Defer, Delete.

## When to use
Daily, in two fixed email sessions (morning + late afternoon). Never "check" email — triage it.

## The 4-D triage
- **Do**: Action takes <5 minutes — do it now, archive.
- **Delegate**: Belongs to someone else — forward/reply with clear ask, archive.
- **Defer**: Requires real work — move to a "Needs Action" folder, add to task system, archive.
- **Delete**: No value — delete or unsubscribe.

## Workflow
```
1. Set a timer: 30 minutes max per session.
2. Open inbox. Sort by newest first.
3. For each email, apply one of the 4-Ds. Never leave an email "unread" as a reminder.
4. Move email out of inbox immediately — archive, Needs Action folder, or trash.
5. At end of session, inbox = 0 unread.
6. Once a week: process the Needs Action folder (turn items into tasks).
```

## Folder setup
```
Inbox (landing zone — should be near-zero after each session)
├── Needs Action (deferred emails with real tasks)
├── Waiting On (emails where you're waiting for a reply)
├── Reference (information you'll need again)
└── Archive (everything else — searchable, no need to sort)
```

## Tips
- Notifications off. Email on your schedule, not everyone else's.
- "Reference" folder is optional — modern search makes folders unnecessary for most people.
- The failure mode is Defer without adding to task system. Deferred emails become a second inbox.
- India context: separate work email from personal from newsletters — three Gmail accounts is standard; use multiple browser profiles.
